### description
- _技术_ : java+jsp+sqlserver2014
- _情景_ : 大三的一次数据库课程设计，室友是游戏王战队联盟组织部部长，有对这个系统的需求。（游戏王是由KONAMI开发的卡牌游戏，是当今世界上最受欢迎、受众面积最广的TCG集换式卡牌游戏之一。它凭着庞大的卡池系统，平衡的游戏环境，二十年不间断地更新以及相关的动漫的影响，吸引了来自全球的大批爱好者。而又随着近些年网络的发展，相关的论坛，社区，由爱好者自发组织的比赛也在国内如火如荼的发展着。
随着游戏王的发展，实卡对战已越来越不满足玩家们的对战需求。所以，在这个网络发达的时代，网上对战已经成为玩家们的新主流，而YGOCORE联盟就是这些玩家的一个最大集合体。YGOCORE联盟包含着管理部门以及各个战队，各个战队包含了包括队长在内的多名队员，管理部门会组织、发布和主持比赛，并对其战报进行统计与管理。然而这些统计和管理工作非常的复杂和繁琐，所以YGOCORE联盟产生了对于游戏王战队信息管理系统的需求。
该系统包含游戏王战队信息的管理和查询、比赛信息的管理和查询、游戏王战队的比赛报名、驱逐名单的管理查询和禁赛名单的管理查询等符合游戏王战队日常管理、比赛发布报名流程、易于实际操作的功能，可以有效的提高游戏王战队信息管理的先进性、减少统计工作量、提高游戏王战队信息管理水平。）
- _任务_ : 我负责完成整个的后端开发。战队信息管理模块，含战队基本信息	管理模块、队员信息管理模块、荣誉信息管理模块、踢馆信息模块；赛程管理模块，含比赛信息管理模块、负责裁判管理模块、战队报名系统模块；黑名单管理模块，含驱逐名单管理模块，禁赛名单管理模块。
- _行动_ : java方面我只是会基础（基本流程控制，面向对象），于是用了几天时间简单的学了jsp，serverlet，在学校图书馆一些实战书的帮助下完成了开发。
- _结果_ : 舍友没有成功连任，这个系统也没人用了，数据库课程设计得了优。
